    • Introduction

    The Biosensor is a non-medical device that is able to detect and calculate the heartrate in BPM and respiration in breaths/minute of a user.
    There are many different circuits and peripherals that are connected to a Ti Tiva C TM4C123GXL board, which runs all of the code and peripherals.

    Heart Rate Monitor

    The heart rate monitor is a simple circuit that uses an IR LED and a photodiode to detect the user's heart rate. The IR LED shines light into the user's finger, and the photodiode detects the amount of light that is reflected back. The amount of light that is reflected back changes as the user's heart pumps blood through their finger, and the photodiode is able to detect this change in light. The code then processes this data and calculates the user's heart rate in BPM.
