    • Introduction

    The Biosensor is a non-medical device that is able to detect and calculate the heartrate in BPM and respiration in breaths/minute of a user.
    There are many different circuits and peripherals that are connected to a Ti Tiva C TM4C123GXL board, which runs all of the code and peripherals. This page goes over all of the functions, and also examples of each except for the respiration monitor (more on that later). This was the very first project I did, so many of the details are fuzzy.
